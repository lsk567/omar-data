# Kalshi Baseline Strategy — Dashboard

Live portfolio data for the baseline macro/economic trading strategy on Kalshi.

**Strategy repo:** [kalshi-baseline](https://github.com/lsk567/kalshi-baseline) — full philosophy, trade cycle process, and trade log with rationale.

## Files

- `portfolio-history.csv` — Time-series snapshots of portfolio value, positions, and unrealized P&L
- `portfolio_snapshot.json` — Detailed position-level snapshot with per-position P&L
- `events.json` — OMAR event log
